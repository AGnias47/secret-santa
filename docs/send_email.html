send_email.gmail_smtp module¶

send_email.gmail_smtp.send_email(sender, recipient, subject, body)¶

Sends an email through a Gmail account using the SMTP protocol

Parameters

  • sender (tuple) – Email address to send from, Password for email address

  • recipient (str) – Email address to send to

  • subject (str) – Subject of the email

  • body (str) – Body of the email

Returns

True upon successful completion, else False

Return type

bool

Notes

Successful execution of the function does not necessarily mean that an email was sent
